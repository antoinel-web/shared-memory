# Morning Memory Sync — Instructions

Ce fichier est lu par le subagent `Morning Memory Sync` à chaque exécution (weekdays 6:00 AM).

---

## Ce que tu dois faire à chaque run

### 1. Lire les logs des deux agents

- Lire `logs/claude-observations.md` → nouvelles entrées depuis hier
- Lire `logs/tasklet-observations.md` → nouvelles entrées depuis hier

### 2. Lire l'inbox

- Lire tous les fichiers `inbox/for-tasklet-*.md` non traités
- Lire tous les fichiers `inbox/for-antoine-*.md` → les inclure dans le digest
- Lire tous les fichiers `inbox/for-claude-*.md` → noter leur existence dans le digest (Claude les lira en session)

### 3. Extraire les patterns d'apprentissage

Dans `logs/claude-observations.md`, chercher les entrées `[FEEDBACK]` et `[SKILL]`.
Ces entrées décrivent comment Antoine travaille et ce qu'il corrige.
Les intégrer dans ta propre compréhension du profil Antoine.

### 4. Écrire le digest

Écrire `logs/digest-YYYY-MM-DD.md` avec ce format :

```
# Morning Memory Digest — YYYY-MM-DD

_Generated by Tasklet Morning Memory Sync_

## Claude Observations (nouvelles)
<liste des entrées OBSERVE/FEEDBACK/SKILL depuis hier, ou "Aucune">

## Tasklet Observations (nouvelles)
<liste des entrées depuis hier, ou "Aucune">

## Inbox — Pour Antoine
<contenu des for-antoine-*.md non traités, ou "Aucun">

## Inbox — Pour Claude (à lire en session)
<liste des fichiers for-claude-*.md en attente, ou "Aucun">

## Amendments en attente d'approbation
<liste des [AMEND] non validés, ou "Aucun">

## Patterns appris aujourd'hui
<résumé des nouveaux [FEEDBACK] détectés — ce que Claude ou Tasklet doit faire différemment>
```

### 5. Traiter l'inbox for-tasklet

Pour chaque fichier `inbox/for-tasklet-*.md` :
- Lire l'instruction
- Exécuter si possible dans ce run
- Marquer `[DONE]` en première ligne du fichier

---

## Règles

- No Slack. Tout va dans le repo.
- Dashboard est la seule surface de review.
- Si un amendment nécessite approbation, le noter dans le digest et NE PAS l'appliquer.
- Si inbox/for-antoine contient une escalade urgente, la mettre EN PREMIER dans le digest.
